Brainwave Project

A lightweight EEG pipeline that turns motor-imagery brain signals into decisions. Built on the BCI Competition IV-2a dataset using CSP feature extraction + SVM classification. Current best window: 1.5–4.0s. Best config: nc=8, C=2.0, cov=concat.

See Results ↓

What this is

The goal: classify Left vs Right hand motor imagery from multi-channel EEG. We segment trials, clean them (notch + 8–30 Hz bandpass), crop to the motor-relevant window, learn CSP spatial filters, and run an SVM on the log-variance CSP features.

  • Dataset: BCI Competition IV-2a (subject A01T)
  • Sampling rate: 250 Hz, Channels: 25
  • Trial length: 4.5 s → cropped to 1.5–4.0 s

Pipeline

  1. Load & segment runs → trials (25×1125 samples)
  2. Filter: notch(60/120 Hz), bandpass(8–30 Hz)
  3. Crop: start=1.5s, dur=2.5s
  4. Features: CSP (nc=8, cov=concat), log-var
  5. Classifier: SVM RBF (C=2.0, gamma=scale)
  6. Validation: 5-fold stratified CV + held-out eval

Code: scripts/load_bci_data.py (training) and scripts/test_model.py (evaluation).

Try the model

The trained pipeline is serialized to models/csp_svm_final.joblib. Load it with joblib.load, apply the same preprocessing and cropping, then run csp.transform → clf.predict. 

from joblib import load
payload = load("models/csp_svm_final.joblib")
csp, clf = payload["csp"], payload["clf"]
Xp = preprocess_and_crop(raw_trials)   # your  (n_trials, 25, time)
Xc = csp.transform(Xp)
yhat = clf.predict(Xc)
